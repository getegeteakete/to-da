# 山荘 透雫 (Sanso Tōda) - Landing Page

福岡の山あいに佇む、築230年の旅館。静謐な時間と自然のなかで、魂の帰る場所を。2027年春、ウェルネスリトリートとして静かに扉を開きます。

A 230-year-old ryokan nestled in the mountains of Fukuoka. A place where the soul returns, amid tranquil time and nature. Opening quietly in Spring 2027 as a wellness retreat.

## Features

- **Bilingual Support**: Complete Japanese/English localization
- **Responsive Design**: Mobile-optimized layout
- **SVG Art**: Custom sumi-e (ink wash) style illustrations
- **Smooth Animations**: Scroll-triggered reveals and parallax effects
- **Single File**: Self-contained HTML with embedded CSS/JS

## Project Details

- **Established**: 1795 (Kansei era)
- **Location**: 711 Iiba, Sawara-ku, Fukuoka 811-1134, Japan
- **Contact**: sansotoda@gmail.com
- **Opening Timeline**: Soft opening March 2027, Grand opening April 2027

## Technology

- Pure HTML/CSS/JavaScript
- Google Fonts (Shippori Mincho, Cormorant Garamond, Noto Serif JP)
- Intersection Observer API
- localStorage for language preference
